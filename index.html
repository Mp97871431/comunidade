Bem estar para todos

Sabemos que existem diversos fatores para manter um bairo, um quarteirão, comunidade unidos, juntamente um desses fatores é o bem estar e segurança para todos. Um momento de laser e socialização para que isso aconteça é muito importante, por isso devemos ter lugares para promover essa socialização, para isso temos que fazer nossa parte cuidando e zelando por esses ambientes que temos a disposição, que são nossas praças por exemplo. Porém tem coisas que não é de nosso alcance e sim de orgões superios para nos disponibilizar de um bom lugar para socialização, mantendoos com bons equipamentos de ezercicios, bons playgraunds para as crianças e segunça para que tudp fique bem, para isso temos esse site que servira para quando indentificarmos a alta de algumas dessas coisas sitadas possamos acionar esses orgões para tomar uma atitude para que sejá sanado o problema.
